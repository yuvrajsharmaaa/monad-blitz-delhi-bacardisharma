# Music Remix Competition Platform on Monad

A full-stack decentralized MVP for a music remix competition platform built on the Monad blockchain.

## Features

- NFT minting for original tracks and remixes
- On-chain voting with dynamic programming optimizations
- IPFS integration for decentralized audio storage
- Prize distribution automation
- Real-time voting interface
- Responsive UI with Next.js

## Tech Stack

- Smart Contracts: Solidity (Hardhat)
- Frontend: Next.js, React, ethers.js
- Storage: IPFS
- Blockchain: Monad

## Project Structure

```
├── contracts/          # Smart contracts
├── frontend/           # Next.js application
├── scripts/            # Deployment scripts
├── test/               # Contract tests
└── ipfs/               # IPFS utilities
```
